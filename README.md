## 🚀 Cara Menggunakan

### 1. Buka Website
- Buka file `html/agama.html` di browser
- Website langsung siap digunakan tanpa instalasi tambahan

### 2. Menambah Tugas
```
1. Scroll ke bagian "Tugas & Jadwal"
2. Ketik topik pembelajaran di input field
3. Klik tombol "Tambah"
4. Tugas akan muncul di daftar di bawah
```

### 3. Mengelola Tugas
- **Edit**: Klik tombol Edit pada tugas, ubah deskripsi, klik Simpan
- **Hapus**: Klik tombol Hapus dan konfirmasi
- **Selesai**: Klik tombol ✓/○ untuk toggle status
- **Filter**: Gunakan filter buttons untuk menyaring tampilan

### 4. Hubungi Kami
- Scroll ke footer
- Klik nomor WhatsApp untuk langsung membuka chat
- Email tersedia juga di bagian Kontak
