## 🕌 About This Project

<div align="center">

**Islamic Learning Hub** adalah platform pembelajaran agama Islam profesional yang dirancang untuk membantu Anda memahami dan memperdalam pengamalan ajaran Islam dengan pendampingan profesional.

## 🎯 How to Use

### 1️⃣ Menambah Tugas
```
1. Scroll ke bagian "Tugas & Jadwal"
2. Ketik topik pembelajaran di input field
3. Klik tombol "Tambah"
4. Tugas akan muncul di daftar!
```

### 2️⃣ Mengelola Tugas
- **✓ Selesai**: Klik tombol centang untuk menandai selesai
- **✏️ Edit**: Klik tombol edit untuk mengubah deskripsi
- **🗑️ Hapus**: Klik tombol hapus untuk menghapus tugas

### 3️⃣ Filter Tugas
Gunakan tombol filter untuk melihat:
- 📋 Semua tugas
- 📖 Sedang dipelajari
- ✅ Selesai
